# 🚲 UniCycle

**A front-end prototype for a university bike-sharing platform focused on sustainable mobility.**

UniCycle is a modern web interface that simulates the experience of a university bicycle-sharing system. The project was designed to showcase the user experience, interface design, and interaction flow of a smart mobility platform connecting university campuses with nearby public transportation stations.

The application demonstrates how users could rent bicycles, monitor station availability, track their rides, earn rewards, and report issues through a clean and intuitive interface.

---

# 📖 Overview

UniCycle was developed as a **front-end prototype** using HTML, CSS, and Vanilla JavaScript.

Rather than implementing a complete backend architecture, the project focuses on designing the complete user experience and simulating the expected behavior of the system through mock data and interactive components.

It serves as a proof of concept for a future full-stack bicycle-sharing platform.

# ⚠️ Project Status

This project is currently a **front-end prototype** intended to demonstrate the design and user experience of a university bike-sharing platform.

The application **does not include**:

* Backend services
* Database integration
* REST APIs
* User authentication
* Real-time synchronization
* Persistent data storage

All information displayed in the application is simulated using JavaScript to represent the expected behavior of a complete production system.
